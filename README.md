# 🔀 CC-Model-Router (CCMR)

**cc-model-router** seamlessly integrates with Claude Code to automatically route your prompts to the optimal Anthropic model. Whether you need the lightning speed of Haiku for quick edits, the balanced power of Sonnet for standard development, or the deep reasoning of Opus for complex architecture, this router puts the right model at your fingertips.

---

## ✨ Features

* **Native Integration:** Built specifically to work alongside Claude Code without clunky workarounds.
* **Smart Routing:** Easily define rules to route tasks based on complexity, context size, or specific tags.
* **Cost & Speed Optimization:** Save tokens and time by defaulting to smaller models for simple tasks and escalating to Opus only when necessary.
* **Zero-Friction Switching:** Change models on the fly without rewriting your base configuration.
